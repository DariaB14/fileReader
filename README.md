N-th Minimum element Finder

REST API для поиска N-го минимального элемента в Excel файле. 

Особенности:
- Чтение файла формата .xlsx, содержащего целые числа.
- Поиск N-го минимального элемента с использованием алгоритма на основе кучи.

Стек: 
Java 17, Spring Boot 3.x, Spring Web MVC, Apache POI, SpringDoc OpenAPI

Запуск:

1. Склонировать репозиторий: git clone 
   cd 
3. Cобрать mvn clean package 
4. Запустить java -jar 


Swagger документация доступна по адресу: http://localhost:8080/swagger-ui.html



Возможные улучшения

Принимать файл через multipart/form-data напрямую (а не путь).
Добавить защиту от повторяющихся чисел (если нужен уникальный N-ый минимум).
Добавить стриминг для больших файлов или чтение постранично.
Логгирование ошибок и обработки исключений через @ControllerAdvice.
